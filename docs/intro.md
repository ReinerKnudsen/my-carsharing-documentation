#  Übersicht

Mit **My Carsharing** steht Dir eine mächtige Webanwendung zur Verfügung, um Dein privates Carsharing zu verwalten. Alle Funktionen sind aus meiner eigenen Erfahrung mit einem mehrjährigen Carsharing-Projekt entstanden. Die benutzerfreundliche Nutzung und die Reduzierung auf wesentliche Funktionen sind Ziel meiner Entwicklung.

## Funktionen 

### Fahrten
Zeichne Deine Fahrten auf. Mit einem einfachen Klick startest Du eine Fahrt und beendest sie genauso einfach. Fahrten werden aufgezeichnet. Du kannst Dir Deine Fahrten und die Fahrten Deiner Gruppe jederzeit anzeigen lassen.

### Buchungen
Buche das Fahrzeug bequem über eine Kalenderfunktion. Jeder Teilnehmer kann die ausstehenden Buchungen sehen und seine eigenen danach einrichten.



### Gruppen
Die Anwendung kann eine oder mehrere Gruppen verwalten. Gruppen können z.B. Familien sein. Eine Gruppe fährt stets auf gemeinsame Rechnung.
