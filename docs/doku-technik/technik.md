# Technik-Stack

## Frontend

- **[React 18](https://react.dev/)** - UI-Framework
- **[TypeScript](https://www.typescriptlang.org/)** - Typsichere Entwicklung
- **[Ionic Framework 8](https://ionicframework.com/)** - Mobile UI-Komponenten
- **[React Router](https://reactrouter.com/)** - Navigation

## Backend & Datenbank

- **[Supabase](https://supabase.com/)** - Backend-as-a-Service
  - PostgreSQL Datenbank
  - Row Level Security (RLS) für Datensicherheit
  - Authentication
  - Storage für Dateien

## Build & Development

- **[Vite](https://vite.dev/)** - Build-Tool
- **[Capacitor](https://capacitorjs.com/)** - Native App-Wrapper (für iOS/Android)
